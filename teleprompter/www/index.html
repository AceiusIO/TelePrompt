TelePrompt is an app that allows you to write speeches and then record them in app.

TelePrompt is a free and open source teleprompter app. It was written in JavaScript, JQuery.js, and RecordRTC.js by AceiusIO.

Additional CSS was written by Emantella on GitHub. This app has been licensed under the GNU Public Licence V3.0, giving you the right to copy,

the source code on GitHub, distrubiute it, and modify it.

Use HTML Markup to style your speech, such as using <strong> and </strong> elements for bold, and <em> and </em> elements for itallics.
